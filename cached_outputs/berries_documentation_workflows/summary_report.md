## Berries

### Documentation Workflow Overview

- Documentation-related features found: 13

### Core Note and Treatment Plan Capabilities

- **Audio Session Recording**: Records in-person and virtual sessions with or without headphones. Multilingual support (English, Spanish, French, Portuguese, more). Accessible via web browser on mobile and desktop.
- **Progress Note Templates**: Built-in templates: SOAP, DAP, BIRP, GIRP, intake, follow-up, couples therapy. Custom template builder with full section control.
- **Custom Template Builder**: Build note templates from scratch with full control over sections. Can create from blank, sample notes, or existing templates. Name, arrange, and customize sections. Share templates with team members.
- **Manual Note Editing**: Direct text editing in generated notes. Manual save required.
- **Note Preferences**: Set tone, length, and clinical focus for generated notes. Customize to match clinician style.
- **Template Switching**: Change note template post-generation. Can regenerate note with different template.
- **Patient Reassignment**: Reassign generated note to different patient if initially assigned incorrectly.
- **Session Assistant Panel**: Right-side panel accessible before, during, and after sessions. Includes Preparation tab, Assistant Chat tab, and Treatment Plan tab. Can stay open during live session.
- **Patient Profile Management**: Centralized patient profiles with diagnosis, session history, treatment plans, and notes. AI-generated patient summaries.
- **Copy to EMR**: Manual copy/paste workflow to transfer notes into any EMR/EHR. No direct integration or API.
- **Treatment Plan Builder**: Create, edit, and regenerate treatment plans. Build from scratch, from sample plans, or auto-generate from session notes. Template customization available.
- **Collaborator Templates**: Add templates shared by colleagues via template codes. Unlock exclusive community-created templates.
- **Multi-Language Session Support**: Conduct sessions in English, Spanish, French, Portuguese, and more. Notes generated in English.

### Relevant AI Features in This Area

- **AI Note Generation**: Fully automated progress note generation from session audio. Transcribes session and generates comprehensive clinical note in ~60 seconds. Supports SOAP, DAP, BIRP, GIRP, and custom templates. Multilingual support (English, Spanish, French, Portuguese).
- **Magic Edit**: Natural language editing tool. Clinician provides instructions (e.g., 'make intervention section more detailed'), and AI rewrites the specified section accordingly.
- **AI Treatment Plan Generation**: Generates treatment plans from initial assessments or session notes. Includes goals, interventions, and progress tracking. Can create from scratch or from sample plans. Fully editable and regenerable.
- **Smart Diagnosis (ICD-10 Suggestions)**: Analyzes session note context and suggests relevant ICD-10 diagnostic codes. Clinician reviews and approves suggestions before adding to note.
- **AI Assistant (Clinical Chat)**: Context-aware HIPAA-compliant chatbot. Uses patient history, diagnoses, and treatment plans to answer clinical questions, suggest interventions, generate follow-up questions, provide billing code recommendations, and offer evidence-based guidance. Available before, during, and after sessions.
- **Session Preparation Summary**: Pre-session AI-generated overview pulling from all prior Berries notes. Highlights safety concerns, therapeutic themes, interventions used, treatment plan progress, and alliance check. Provides structured prep view for clinicians.
- **Patient Letter Generation**: Generates client instructions and letters from session notes. Template-based generation for post-session communication. Separate output from progress notes.
- **Quick Edit (Section Regeneration)**: Allows clinicians to regenerate specific sections of notes rather than entire note. Targeted AI re-generation for efficiency.
